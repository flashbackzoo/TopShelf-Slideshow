TopShelf Slideshow

Nested Slideshow Demo

About:

TopShelf Slideshow is a UI plugin for jQuery.
Use it to create customizable user interfaces for your website.

This demo shows how TopShelf Slideshow can be used to create nested slideshows.
The outer slideshow is the main page navigation. The inner slideshow displays some Acid Wasp wallpapers.
